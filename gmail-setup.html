Gmail Setup - Disaster Management System

Direct Gmail Integration

This page will help you set up the Disaster Management System to send emails directly from Gmail instead of using EmailJS as a middleman.

Important Security Note:

You will need to use an "App Password" for your Gmail account, not your regular password. This is a security feature of Google.

Steps to Create a Gmail App Password:

  1. Go to your Google Account Security Settings
  2. Make sure 2-Step Verification is enabled for your account
  3. Scroll down to "App passwords" (or search for it)
  4. Select "Mail" as the app and "Other" as the device
  5. Enter "Disaster Management System" as the name
  6. Click "Generate"
  7. Google will display a 16-character password - copy this password
  8. Use this password in the form below (do not use your regular Gmail password)

Configure Gmail Settings

After Saving Settings:

  1. Restart the backend server using the command: npm start in the server directory
  2. Test the email functionality by creating a new alert or sending an existing alert
  3. Check your Gmail "Sent" folder to confirm emails are being sent directly from your Gmail account
